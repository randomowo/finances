# finances terminal app

#### dependencies:
* __python >= 3.7.2__

### Instalation:
    
    git clone --recursive https://github.com/randomowo/finances.git

    cd finances

    chmod 776 install.sh

    ./install.sh

### Uninstalling:
    chmod 776 uninstall.sh

    ./uninstall.sh
